### Configure the mcp.json file

For workspace configuration, create `.vscode/mcp.json`. For user profile configuration, run `MCP: Open User Configuration`.

```json
{
  "servers": {
    "my-mcp-server": {
      "type": "stdio",
      "command": "my-command",
      "args": []
    }
  }
}
```

### MCP server trust

When you start an MCP server, VS Code prompts you to confirm that you trust the server and its configuration before running it.

### Troubleshoot MCP servers

Run `MCP: List Servers`, select the server, and then select `Show Output` to inspect the server log.
