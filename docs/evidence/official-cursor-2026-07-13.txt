Cursor supports local `stdio` servers.

Create `.cursor/mcp.json` in your project for project-specific tools, or `~/.cursor/mcp.json` for tools available across projects.

```json
{
  "mcpServers": {
    "server-name": {
      "command": "node",
      "args": ["server.js"],
      "env": {}
    }
  }
}
```

MCP tools are listed under Available Tools and can be enabled or disabled before agent use.
