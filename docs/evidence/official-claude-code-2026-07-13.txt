### Option 3: Add a local stdio server

```bash
claude mcp add [options] <name> -- <command> [args...]
```

The `--` (double dash) separates Claude's CLI flags from the command and arguments passed to the MCP server. Without it, Claude may interpret server flags as its own.

Use `claude mcp get <name>` to inspect a server and `claude mcp remove <name>` to remove it.
